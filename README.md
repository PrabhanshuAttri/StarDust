# StarDust

A pseudo Front End Application which would let the users search for different movies online.

This repo is named StarDust after the codename of [Death Star](http://starwars.wikia.com/wiki/Stardust).

## Getting Started
Just open index.html with internet connection.

Note: This repo requires polyfills.
